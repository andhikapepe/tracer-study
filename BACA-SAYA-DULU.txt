## Aplikasi ini dibuat dengan :
1. PHP Version 7.3.11
2. CodeIgniter-3.1.11-0
3. SMTP Gmail
4. Admin template AdminBSB
5. Landing Page template MDBootstrap
6. free images support by freepik.com

## Cara install
1. copy folder ini di folder htdocs jika menggunakan xampp, atau di public_html server web anda
2. setting database di "application/config/database", sesuaikan dengan pengaturan anda.
3. buka settingan phpmyadmin di browser kemudian import database.
4. buka browser, ketikkan alamat sesuai yang anda lakukan di langkah nomor 1. *misal : localhost/nama-folder (jika di localhost)
5. Aplikasi siap digunakan
